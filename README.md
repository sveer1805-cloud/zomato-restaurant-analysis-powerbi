# Zomato Restaurant Analysis Dashboard

## Project Overview

This project presents an interactive Power BI dashboard built using the Zomato Restaurants Dataset. The dashboard provides insights into restaurant distribution, customer ratings, cuisines, online delivery services, and table booking availability across multiple countries and cities.

## Business Objective

To analyze restaurant performance and customer preferences to support data-driven decision-making in the food and hospitality industry.

## Key Insights

- Analyzed 818 restaurants across 77 cities and 15 countries.
- Evaluated average restaurant ratings and customer voting patterns.
- Compared cuisine popularity across different regions.
- Analyzed online delivery and table booking adoption.
- Visualized restaurant distribution using geographical maps.

## Dashboard Features

- Country-wise Restaurant Analysis
- City-wise Restaurant Distribution
- Average Rating Analysis
- Cuisine-wise Performance
- Online Delivery Analysis
- Table Booking Analysis
- Interactive Filters and Slicers
- Geographical Visualization
